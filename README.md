# Python Data Scraper

A Python web scraper that extracts quotes, authors, and tags from a website and stores the data in CSV format.

## Features
- HTTP requests using requests
- HTML parsing using BeautifulSoup
- Pagination support
- CSV data export
- Error handling
- Timestamp logging

## Installation

pip install -r requirements.txt

## Run

python data_scraper.py

## Output

The script generates a quotes.csv file containing:
- Quote text
- Author
- Tags
- Timestamp of scraping
